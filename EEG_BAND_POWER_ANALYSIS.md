# Muse S 脑电频段能量分析报告

## 问题描述

我们用 Muse S 头带采集 5 分钟数据后，计算五个频段的相对功率，结果如下：

| 频段 | 频率范围 | 带宽 | 我们的结果（相对功率） | Muse 官方 APP（dB） |
|------|---------|------|---------------------|-------------------|
| Delta | 0.5–4 Hz | 3.5 Hz | ~0.05 | ~22 dB（最高） |
| Theta | 4–8 Hz | 4.0 Hz | ~0.05 | ~15 dB |
| Alpha | 8–13 Hz | 5.0 Hz | ~0.08 | ~15 dB |
| Beta | 13–30 Hz | **17 Hz** | **~0.40（最高）** | ~8 dB |
| Gamma | 30–45 Hz | **15 Hz** | **~0.35** | ~5 dB（最低） |

两边的频段排序**完全颠倒**：我们 Beta 最高，官方 Delta 最高。

---

## 根因分析

### 1. 纵坐标含义完全不同

| | 我们的值 | Muse 官方 APP |
|---|---|---|
| **物理含义** | 相对功率（无单位，0–1） | 绝对功率谱密度（dB, 参考 1 μV²/Hz） |
| **计算公式** | `band_power / total_power` | `10 × log₁₀(PSD_band / 1μV²/Hz)` |
| **受总功率影响** | 是——总功率是分母 | 否——每个频段独立 |

我们的 0.4 意思是"该频段占总功率的 40%"。官方的 22 dB 意思是"该频段的绝对功率是 1μV²/Hz 的约 158 倍"。

### 2. 核心问题：信号被宽带噪声主导

这是最关键发现。我们的数据呈现出**白噪声频谱特征**：

```
如果信号是纯白噪声（PSD 平坦）：
  相对功率 ∝ 频段带宽

  Delta: 3.5 Hz → 3.5/44.5 =  7.9%  ← 我们测到 ~5%
  Theta: 4.0 Hz → 4.0/44.5 =  9.0%  ← 我们测到 ~5%
  Alpha: 5.0 Hz → 5.0/44.5 = 11.2%  ← 我们测到 ~8%
  Beta: 17.0 Hz → 17/44.5 = 38.2%  ← 我们测到 ~40% ✓
  Gamma:15.0 Hz → 15/44.5 = 33.7%  ← 我们测到 ~35% ✓
```

**我们的结果与"纯白噪声"的理论值高度吻合**。这说明采集到的信号中，真正的脑电成分被噪声淹没了。

### 3. 噪声来源

Muse S 干电极（无导电膏）本身信噪比就低。主要噪声源：

| 噪声类型 | 频率分布 | 影响 |
|---------|---------|------|
| **肌电（EMG）** | 20–200 Hz，宽带 | Beta/Gamma 频段的主要污染源。说话、眨眼、面部微表情都会产生 |
| **电极阻抗噪声** | 全频段 | 干电极接触不良时热噪声大幅增加 |
| **放大器底噪** | 宽带白噪声 | Muse S 消费级芯片的固有噪声 |
| **工频干扰** | 50 Hz 及谐波 | 虽然窄带，但谐波会泄露到相邻频段 |

健康的脑电信号遵循 1/f 规律：频率越高，功率越低。Delta 应该是 Beta 的 10–100 倍。但当 EMG + 电极噪声远大于脑电时，1/f 规律被白噪声覆盖，相对功率就变成了"谁的频段更宽"。

### 4. 为什么 Muse 官方 APP 结果正常

| | 我们的做法 | Muse 官方 APP |
|---|---|---|
| **计算窗口** | 1 秒（256 采样点） | 可能 2–4 秒或多窗口平均 |
| **频谱分辨率** | 2 Hz/bin（nperseg=128） | 更高分辨率 |
| **伪迹剔除** | 无 | 自动检测并丢弃 EMG/眼动/运动污染的 epoch |
| **纵坐标** | 相对功率（比例） | 绝对 dB |
| **电极质量检测** | 无 | 实时检测阻抗，差时提示用户调整 |

最关键的是 **伪迹剔除**。官方 APP 会先判断这几秒的数据是否"干净"（低 EMG、低眼动、电极接触良好），只对干净的 epoch 做频段分析。我们不做这一步，所以即使数据里有大量噪声，也算进去了。

## 验证方法

可以用以下实验验证上述分析：

### 实验 1：白噪声检验
闭眼静坐 2 分钟（减少眼动和面部肌肉活动），看频段排序是否改善。如果是 EMG 主导，闭眼静坐后 Beta/Gamma 应该明显下降。

### 实验 2：查看原始波形
用 `lsl_viewer.py` 看原始 EEG 波形。如果波形非常"毛"（高频抖动大），就是噪声主导。干净的脑电波形应该相对平滑。

### 实验 3：对比 PSD 图形
在报告生成器里添加一个开关，改为输出**绝对 dB 值**而非相对功率。绝对 dB 下，健康的脑电应该显示 Delta/Theta 最高，与我们现在的相对功率排序完全不同。

---

## 建议改进方案

### 短期（快速修正）
1. **输出绝对 dB**：`10 × log₁₀(band_power)` 替代相对功率。即使有噪声，1/f 规律在绝对 dB 下也会更明显。
2. **增大窗口**：从 1 秒改为 4 秒（1024 点），`nperseg=256`，频谱分辨率从 2 Hz 提升到 1 Hz。
3. **多窗口平均**：算 3–5 个窗口取中位数，抑制瞬时噪声。

### 中期（提高信噪比）
4. **添加 EMG 检测**：如果 Beta/Gamma 相对功率 > 50%，标记该 epoch 为"高噪声"，不纳入平均。
5. **工频陷波**：50 Hz notch filter 去除电力线干扰。

### 长期
6. **电极质量提示**：计算信号 RMS 和偏置，实时提示用户调整头带。
7. **采集端预处理**：在 Android 端做简单的伪迹检测，只上传干净的 epoch。

---

## 结论

**两边的差异不是"谁的算法对"，而是数据质量和使用方式的根本不同**：
- 我们算的是带噪声的原始信号，相对功率被宽带噪声主导，Beta 最宽所以最高
- 官方 APP 先筛掉噪声 epoch，只对干净脑电做分析，所以呈现正确的 1/f 频谱

要得到和官方接近的结果，最关键的改进是**伪迹检测和剔除**，其次是用**绝对 dB 纵坐标**。
